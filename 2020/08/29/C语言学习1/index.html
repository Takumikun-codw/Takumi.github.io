编程练习5.11.9

编写一个程序，要求用户输入一个华氏温度。程序应读取double类型的值作为温度值，并把该值作为参数传递给一个用户自定义的函数Temperatures（）。该函数计算摄氏温度和开式温度，并以小数点后面两位数字的精度显示3种温度。要使用不同的温标来表示这3个温度值。下面是摄氏温度转开式温度的公式：（略）

Temperatures（）函数中用const创建温度转换中使用的变量。在main（）函数中使用一个循环让用户重复输入温度，当用户输入q或其他非数字时，循环结束。

我第一次编写的程序：

#include <stdio.h>

int main()
{
double Fa;
printf(“Enter a temperature in Fahrenheit:”);
scanf(“%f”, &Fa);
while(Fa > 0) {
Temperatures(Fa);
printf(“Enter next:”);
scanf(“%f”,&Fa);
}
getchar();
return 0;
}
void Temperatures(double n)
{
const double C_TO_K = 273.16;
const double F_TO_C = 32.0;
double Ce, Ke;
Ce = 5.0/9.0*(n - F_TO_C);
Ke = Ce + C_TO_K;
printf(“%.2f Fahrenheit is %.2f in Celsius and %.2f in Kelvin.\n”, n, Ce, Ke);
return 0;

}

没有完全满足题设要求。因为我发现输入字符不会终止循环，反而会进入无限循环。

原因出在while后面的条件。因为之前做过很多要求输入正值的练习，所以这里欠考虑了。

改进后代码：

#include <stdio.h>
#include <stdlib.h>

int main()
{
double Fa;
printf(“Enter a temperature in Fahrenheit:”);
while(scanf(“%f”, &Fa) == 1){
Temperatures(Fa);
printf(“Enter next:”);
scanf(“%f”,&Fa);
}
getchar();
return 0;
}
int Temperatures(double Fahrenheit)
{
const double C_TO_K = 273.16;
const double F_TO_C = 32.0;
double Ce, Ke;
Ce = 5.0/9.0*(Fahrenheit - F_TO_C);
Ke = Ce + C_TO_K;
printf(“%.2f Fahrenheit is %.2f in Celsius and %.2f in Kelvin.\n”, Fahrenheit, Ce, Ke);
return 0;
}

这里利用了scanf（）函数的返回值。scanf（）函数返回读取数据的数量，所以输入数字则返回1，读取字符等非数字则返回0，从而结束循环。
